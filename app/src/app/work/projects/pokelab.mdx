## Overview

A browser-based 3D Pokeball customizer built with Next.js, React Three Fiber, Three.js, and Tailwind CSS. Pick colors, finishes, patterns, lighting, and backdrops, then share the design with a URL or download it as a PNG.

## Key Features

- Interactive 3D Pokeball preview with orbit controls
- Custom colors for the top shell, bottom shell, center band, button, and button highlight
- Finish options: glossy, matte, and metallic
- Pattern options: classic, stripe, split, and accent
- Lighting presets: studio, sunset, and night
- Backdrop presets: graphite, sky, mint, and plum
- Shareable URLs backed by query parameters
- PNG export from the WebGL canvas
- Responsive viewport tuning for mobile and desktop

## Technologies Used

- [Next.js 16](https://nextjs.org/) App Router
- [React 19](https://react.dev/) and [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/getting-started/introduction) and [Three.js](https://threejs.org/) for 3D rendering
- [@react-three/drei](https://docs.pmnd.rs/drei/getting-started/installation) for 3D helpers and abstractions
- [Tailwind CSS](https://tailwindcss.com/) for styling
- [Lucide Icons](https://lucide.dev/) for UI icons

## Challenges and Learnings

This project was a fun opportunity to experiment with 3D rendering in the browser and explore how to create an engaging user experience around it. One of the biggest challenges was optimizing the performance of the 3D rendering while still providing a high level of visual fidelity and interactivity. I had to carefully balance the complexity of the 3D model, the number of lights and shadows, and the responsiveness of the controls to ensure a smooth experience across different devices. I also had to learn how to use React Three Fiber and Three.js effectively, which involved a lot of trial and error as I figured out how to structure the 3D scene, manage state, and handle user interactions.

## Outcome

The project was a great success and received positive feedback for its fun and intuitive interface, as well as the quality of the 3D rendering. It was a rewarding experience to see users create and share their own custom Pokeballs, and it inspired me to continue exploring the possibilities of 3D in web development. The project is still live and available for anyone to try out and customize their own Pokeball!
