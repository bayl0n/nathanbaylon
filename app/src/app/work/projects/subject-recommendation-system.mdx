## Overview

In this project, fellow Canva Software Engineer, Jacob Gad, and I built a Subject Recommendation System (SRS) using the T3 stack as a deliverable for System Design and Software at UTS. The SRS is designed to provide personalized subject recommendations based on user preferences and behavior.

## Key Features

- **Subject Recommendation**: The system analyzes user data to recommend subjects that align with their interests and learning goals.
- **Admin Tooling**: Established protected routes for admin users to manage subjects.
- **Modern UI**: Utilised Tailwind CSS and animation frameworks to create a clean and responsive user interface that enhances the user experience.

## Technologies Used

- **React and Next.js**: For building the frontend and managing routing.
- **TypeScript**: For ensuring type safety and improving code maintainability.
- **Tailwind CSS**: For styling the application with a utility-first approach.
- **Prisma**: For database management and ORM.
- **Vercel**: For deployment and hosting.

## Challenges and Learnings

One of the biggest challenges in this project was implementing the recommendation algorithm to provide accurate and relevant subject suggestions. We experimented with various approaches, including collaborative filtering and content-based filtering, to find the best fit for our use case. At first, we looked at more complex machine learning approaches such as k-nearest neighbors, but ultimately decided to implement a simpler algorithm that could be easily maintained and scaled as needed given the timeframe for the project.

Working on this project taught me the challenges outside of the technical implementation, such as project management and collaboration. We had to coordinate our efforts and consider trade offs of implementation given the time constraints of the project.

In addition to that, this was my first time collaborating on a project, which made me become familiar with the pull request process and carried me through the entire software development lifecycle, from planning and design to implementation and deployment. This experience has been invaluable in preparing me for future collaborative projects and has strengthened my ability to work effectively in a team environment.

## Outcome

We successfully built and deployed the Subject Recommendation System, which is currently still live. The system has received positive feedback for its user-friendly interface and recommendations and as a result, we received a High Distinction for the course.
