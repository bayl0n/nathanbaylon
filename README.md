# Nathan Baylon

Personal portfolio for [nathanbaylon.com](https://nathanbaylon.com), showcasing my work as a software engineer building full-stack products, modernising systems, and applying practical AI tooling.

## Featured Work

### Rakan: A Personal Finance Management App

A personal finance dashboard for estimating Australian take-home pay, turning income into a practical budget, and tracking expenses against the plan.

### Coral RAG Agent

An AI agent built for Coral Active to help employees retrieve information from internal Confluence documentation and generate Coral SDK code snippets.

### Subject Recommendation System

A collaborative university project built with the T3 stack to recommend subjects based on user preferences and behaviour.
